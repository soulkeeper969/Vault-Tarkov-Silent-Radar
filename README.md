---

## 🛰️ Features

- Stealth mode (no window)  
- CSV data export  
- Enemy movement logger  
- Pinned range scanner  
- Boot with system tray

---

## 🛠️ Instructions

1. Run Tarkov  
2. Launch `Loader.exe`  
3. Data logs start automatically
